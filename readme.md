# Con Magui a Disney – Travel Planner

Landing page orientada a generación de leads y cierre de ventas por WhatsApp
para servicios de viajes Disney, Universal, cruceros y USA.

## Stack
- HTML5
- Tailwind CSS (CDN)
- JavaScript básico

## Objetivo
Convertir visitantes en contactos calificados (leads) y ventas.

## Estado
En desarrollo – optimización de conversión y embudos.
