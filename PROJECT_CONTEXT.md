# Project Context

## Marca
Con Magui a Disney es una marca cercana, empática y experta.
Debe transmitir experiencia real, no venta agresiva.

## Público objetivo
Familias latinoamericanas que viajan a Disney.
Usuarios no técnicos, que buscan confianza y acompañamiento.

## Objetivo principal
Generación de leads y consultas calificadas.
La web debe guiar al usuario de forma clara y simple.

## Tono y comunicación
- Cercano
- Claro
- Tranquilizador
- Profesional sin ser frío

## Prioridades
1. Confianza
2. Claridad
3. Conversión
4. Performance
