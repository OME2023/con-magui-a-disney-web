# Entorno de desarrollo – Con Magui a Disney

## Tipo de proyecto
Landing page estática orientada a generación de leads.

## Stack
- HTML5
- Tailwind CSS (CDN)
- JavaScript vanilla
- Sin build step

## Requisitos del entorno
- Visual Studio Code
- Git
- Navegador moderno (Chrome / Edge / Safari)

## Compatibilidad
- Windows
- macOS

## Ubicación del proyecto
Carpeta sincronizada en Dropbox para acceso desde múltiples dispositivos.

## Reglas del entorno
- No usar dependencias locales (node_modules)
- No usar rutas absolutas
- Mantener nombres de archivos en minúscula
- Git obligatorio antes de cambiar de equipo

## Visualización local
Abrir index.html directamente o usar Live Server (opcional).
